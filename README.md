# css-school

Pequenos estudos de CSS com foco em visual, acabamento e testes rápidos de interface.

A ideia deste repositório é reunir exemplos simples, diretos e visuais que mostram como detalhes pequenos no CSS podem fazer bastante diferença no resultado final.

## O que você vai encontrar aqui

- efeitos visuais
- testes de layout
- estudos de componentes
- snippets pequenos de CSS
- exemplos prontos para abrir no navegador
