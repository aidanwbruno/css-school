o cursor vira parte da tela

Movimento suave, luz, cor e profundidade para deixar a interface mais viva sem pesar demais no visual.

HTML
<div class="blobs">
  <div class="blob blob-1"></div>
  <div class="blob blob-2"></div>
  <div class="blob blob-3"></div>
</div>

<h1>o cursor vira
  <span class="soft">parte da tela</span>
</h1>
CSS
h1 {
  color: transparent;
  background: radial-gradient(
    circle 280px at var(--mx) var(--my),
    #fff, #b794ff, #78e6ff, #f4a8ff
  );
  background-clip: text;
  -webkit-background-clip: text;
}

.blob {
  mix-blend-mode: screen;
  will-change: transform;
}
GSAP
window.addEventListener("mousemove", (event) => {
  gsap.to(".blob-1", { x: event.clientX, y: event.clientY });
  gsap.to(".blob-2", { x: event.clientX, y: event.clientY });
  gsap.to(".blob-3", { x: event.clientX, y: event.clientY });
});
